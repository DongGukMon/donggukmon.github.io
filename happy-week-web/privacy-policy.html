개인정보 처리방침

행복을사다 (HappyWeek)

1. 개인정보의 수집 및 이용 목적

행복을사다(이하 "앱")는 다음의 목적을 위해 최소한의 개인정보를 수집·이용합니다.

수집 목적 수집 항목 보유 기간
맞춤형 광고 제공 광고 식별자 (iOS IDFA / Android GAID) 앱 삭제 시 또는 사용자 추적 거부 시까지
자동 로그인 (동행복권) 동행복권 아이디·비밀번호, PIN 번호 기기 내 안전한 저장소(Keychain/Keystore)에 보관, 로그아웃 시 즉시 파기

2. 광고 식별자 수집 및 제3자 제공

앱은 Google AdMob을 통해 광고를 제공하며, 이를 위해 광고 식별자를 수집합니다.

  • iOS: App Tracking Transparency(ATT) 프레임워크를 통해 사용자 동의를 받은 후 IDFA를 수집합니다.
  • Android: Google 광고 ID(GAID)를 수집합니다.
  • 제공받는 자: Google LLC (Google AdMob)
  • 제공 목적: 사용자 맞춤형 광고 제공 및 광고 성과 측정
  • 추적 거부 시: 사용자가 추적을 거부하면 비개인화 광고가 제공되며, 광고 식별자는 수집되지 않습니다.

사용자는 기기 설정에서 언제든지 추적 허용 여부를 변경할 수 있습니다.

  • iOS: 설정 > 개인정보 보호 및 보안 > 추적
  • Android: 설정 > Google > 광고 > 광고 ID 삭제

3. 자동 로그인 자격증명 저장

앱은 사용자가 자동 로그인을 선택한 경우, 동행복권 사이트 로그인에 필요한 아이디와 비밀번호를 기기 내 안전한 저장소에 암호화하여 저장합니다.

  • iOS: Keychain Services를 사용하여 암호화 저장 (Keychain 특성상 앱 삭제 후에도 데이터가 잔존할 수 있습니다.)
  • Android: Android Keystore를 사용하여 암호화 저장 (앱 삭제 시 자동으로 삭제됩니다.)
  • 저장된 자격증명은 외부 서버로 전송되지 않으며, 기기 내에서만 사용됩니다.
  • PIN 번호는 자동 로그인 인증 수단으로만 사용되며 동일한 방식으로 안전하게 저장됩니다.
  • 로그아웃 시 저장된 모든 자격증명과 PIN이 즉시 삭제됩니다.

4. 개인정보의 파기

앱은 수집한 개인정보의 보유 기간이 경과하거나 처리 목적이 달성된 경우 지체 없이 해당 정보를 파기합니다.

  • Android: 앱 삭제 시 기기에 저장된 모든 자격증명 및 PIN이 자동으로 삭제됩니다.
  • iOS: Keychain 특성상 앱 삭제 후에도 데이터가 잔존할 수 있습니다. 로그아웃 후 앱을 삭제하시는 것을 권장합니다.
  • 로그아웃 시 Secure Store에 저장된 모든 데이터가 즉시 삭제됩니다.

5. 이용자의 권리

  • 사용자는 언제든지 ATT 추적 동의를 철회할 수 있습니다.
  • 사용자는 앱 내 로그아웃을 통해 저장된 자격증명을 삭제할 수 있습니다.
  • 사용자는 앱을 삭제하여 개인정보를 제거할 수 있습니다. (iOS의 경우 Keychain 데이터가 잔존할 수 있으므로, 삭제 전 로그아웃을 권장합니다.)

6. 개인정보의 안전성 확보 조치

앱은 개인정보의 안전성을 확보하기 위해 다음 조치를 취하고 있습니다.

  • iOS Keychain / Android Keystore를 활용한 하드웨어 수준 암호화 저장
  • 외부 서버로의 자격증명 전송 없음 (기기 내 처리)
  • PIN 기반 접근 제어를 통한 무단 사용 방지
